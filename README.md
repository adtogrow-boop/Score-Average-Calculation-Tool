# Score Average Calculation Tool

A simple cricket tool to calculate:

- Required run rate
- Balls remaining
- Runs needed to win

## Demo
After enabling GitHub Pages, the tool will be available here:
https://YOUR-USERNAME.github.io/Score-Average-Calculation-Tool

## How to use
1. Enter **Runs Required**
2. Enter **Overs Remaining** (use format like `4.3` for 4 overs 3 balls)
3. Click **Calculate**  
The tool will show:
- Required Run Rate  
- Total balls remaining  
- Runs needed to win
